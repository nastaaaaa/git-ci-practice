# Система управления заявками на расходы
## gos.obr.budget.control

### Описание проекта
Информационная система для автоматизации планирования, учета и контроля расходов 
бюджетной образовательной организации с интеграцией в 1С:Бухгалтерия государственного учреждения.

### Основные функции
- Учет лимитов бюджетных обязательств
- Проверка КБК (кодов бюджетной классификации)
- Формирование и согласование заявок на расходы
- Контроль исполнения лимитов
- Ведение истории и версий заявок
- Интеграция с 1С:БГУ

### Проектные данные
- Заказчик: Бюджетное образовательное учреждение ДПО
- Сроки: 01.01.2022 - 01.07.2022
- Бюджет: 700 000 рублей
